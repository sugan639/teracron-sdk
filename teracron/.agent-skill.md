# Teracron SDK — Agent Skill Reference (Root)

> Quick reference for AI agents interacting with the Teracron Python SDK.
> For the full tracing reference, see `teracron/tracing/.agent-skill.md`.

---

## Quick Start

```bash
# 1. Login
teracron-agent login --api-key tcn_...

# 2. Check auth
teracron-agent whoami

# 3. Run the agent
teracron-agent run

# 4. Query events
teracron-agent events --status=failed --json

# 5. Diagnose a failure
teracron-agent trace <TRACE_ID> --json
teracron-agent simulate <TRACE_ID> --format=markdown

# 6. Get curl examples
teracron-agent curl-example
```

---

## CLI Commands

| Command | Description |
|---|---|
| `teracron-agent` | Run metrics agent (default) |
| `teracron-agent run` | Run metrics agent (explicit) |
| `teracron-agent login` | Authenticate and store API key |
| `teracron-agent logout` | Wipe stored credentials |
| `teracron-agent whoami` | Show current auth status |
| `teracron-agent events` | Query recent workflow events |
| `teracron-agent workflows` | List workflow run summaries |
| `teracron-agent trace <id>` | Fetch a full trace span tree |
| `teracron-agent simulate <id>` | Replay a failed trace |
| `teracron-agent curl-example` | Print curl command examples |

All commands support `--json` for machine-readable output.
All commands support `--api-key` to override stored credentials.

## Environment Variables

```bash
TERACRON_API_KEY          # Required (or use `teracron-agent login`)
TERACRON_DOMAIN           # Default: www.teracron.com
TERACRON_DEBUG            # "true" or "1" for debug logging
TERACRON_TRACING_ENABLED  # Default: true
TERACRON_TRACE_SAMPLE_RATE # 0.0–1.0 (default: 1.0)
TERACRON_TRACE_EMIT_EVENTS # "true" to emit workflow events
```

## Curl Access (for AI Agents)

```bash
export TERACRON_API_KEY="tcn_..."

# Failed events
curl -s -H "Authorization: Bearer $TERACRON_API_KEY" \
  "https://www.teracron.com/v1/events?status=failed&limit=10"

# Trace detail
curl -s -H "Authorization: Bearer $TERACRON_API_KEY" \
  "https://www.teracron.com/v1/traces/<TRACE_ID>"

# Workflow summaries
curl -s -H "Authorization: Bearer $TERACRON_API_KEY" \
  "https://www.teracron.com/v1/workflows?limit=20"
```
